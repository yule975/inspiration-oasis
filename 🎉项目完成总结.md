## 🚀 **立即启动使用**

### 快速部署命令
```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
pnpm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件配置数据库和其他服务

# 4. 初始化数据库
pnpm db:generate
pnpm db:push
pnpm db:seed

# 5. 启动开发服务
pnpm dev

# 6. 运行测试验证
pnpm test

# 🎉 系统就绪!
# API服务: http://localhost:3001
# 健康检查: http://localhost:3001/health
# API文档: http://localhost:3001/api
# 监控面板: http://localhost:3001/api/monitoring/status
```
